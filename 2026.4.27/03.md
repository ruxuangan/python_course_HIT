## 1. Data Structure 
### [1] tuple = (1,"blue",3)
- ordered
- immutable (cannot change)
- have length  ```len(tuple)``` 
- start with 0  ```tuple[1] 是第二个数```
- iterable(可参与循环)  ```for a in tuple```
- 检查元素是否在其中  ```if "red" in tuple```
  
### [2] list = [1,"milk",3]
- mutable  
- ```.split(这里填string的分隔符)``` turn string into list
- ```.insert(1,"C")``` 将C插入指定位置
  ```.pop(1)``` 取出指定位置元素
  ```.append("C")``` 将一个元素D插入最右侧
  ```.extend()``` 将几个元素插入最右侧，多个元素需写成list形式
- sum, max, min均可对list使用
- ```.sort()``` 将列表中数字从小到大排列，可使用```reverse = True``` 实现降序，```key = len``` 实现自定义要求排序
> sorted() 函数也能实现排序，但会生成新列表且不改变原列表
- 由于list是mutable的，修改一个列表会同时修改与它相同的列表。使用```large_cats = animals[:]```，在赋值变量时赋予一个新的列表副本


### [3] dictionary = {"a":1, "b":2 ,"c":3} {1}
- key : value  
- ```dictionary[key]``` 输出value
- ```dictionary[key] = value``` 可以直接增加或修改内容
- ```del dictionary[key]``` 删除内容
- iterable，```for a in capitals:``` 中a是key,```for b,c in capitals.items():``` 以同时获取key和value

### [4] set = {"a", "b", "c"}{1}
- unordered
- unindex
- immutable

numbers = (1, 2, 3, 4, 5)
