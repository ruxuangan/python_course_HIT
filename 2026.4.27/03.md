## 1. Data Structure 
### [1] tuple = (1,"blue",3)
- ordered
- immutable (cannot change)
- have length  ```len(tuple)``` 
- start with 0  ```tuple[1] 是第二个数```
- iterable(可参与循环)
  
### [2] list = [1,"milk",3]
- mutable  
- .split() turn string into list
- ```.insert(1,"C")```将C插入指定位置
  ```.pop(1)``` 取出指定位置元素
  ```.append("C")``` 将一个元素D插入最右侧
  ```.extend()``` 将几个元素插入最右侧
- 使用: ，在赋值变量时赋予一个新的列表副本

### [3] dictionary = {"a":1, "b":2 ,"c":3} {1}
- key : value  

### [4] set = {"a", "b", "c"}{1}
- unordered
- unindex
- immutable
