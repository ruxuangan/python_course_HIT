## 1. 条件语句  
- ### if 判断：  
  ```
  if [condition1] :
    bit of code  
  elif [condition2] :
    bit of code  
  else :  
    bit of code  
  ```  
  > 如果condition1满足，则不会判断condition2  

## 2. write function   
```  
def func(x,y):        # function signature  
    z = x*y
    print(z)          # function body  
```  
> 函数内的变化对外界不可见，此例中z的取值在函数外不变  

## 3. loops  
- ### for 循环  
  ```  
  for i in range(a,b):     # b是截止位，即不会运行b
    bit of code  
  ```  
  continue: 结束当前循环节，进行下一个循环  
  break: 结束所有循环  
  pass: 占位符，无意义  

- ### while 循环  
  ```  
  while [condition]:  
    do whatever  
  ```
