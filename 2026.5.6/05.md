## 1. class
**用处** ： 储存一大类物品，并能定义他们的行为  
**示例** : 
```python
class Employee:  # 常用大写
    def __init__(self, name, age):
        self.name = name
        self.age = age

    # Instance method
    def description(self):
        return f"{self.name} is {self.age} years old"
    
    # Another instance method
    def update_age(self, new_age):
        self.age = new_age

    # 定义print内容
    def __str__(self):
        return f"{self.name} is {self.age} years old"
emp1 = Employee("David", 23)
emp2 = Employee("Alice", 35)
```  

## 2. files  
### step1 : open a file  
```my_file = open(file='file name', mode='w')```  
- "w": write 创建或重写一个文档并写入内容  
- "r": read 读文档
- "a": append 创建或在文档末尾加入内容，不会覆盖原有内容  

### step2 : write/read a file
```my_file.write('Hello file!')``` *不会自动换行，需使用'/n'*  
```file_output = my_file.read()``` *输出文档内容*
```file_output = my_file.readline()``` *输出文档第一行*
```file_output = my_file.readlines()``` *将文档每行存入列表中*  

### step3 : close the file
```my_file.close()```
